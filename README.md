# 살까말까 🛒

충동 구매를 방지하는 위시리스트 앱

사고 싶은 물건을 바로 사지 않고 담아두면서, 시간이 지남에 따라 관심도가 자연스럽게 낮아지는 것을 시각화합니다.

## 핵심 기능

- **위시리스트** — 사고 싶은 것을 담아두고 왜 필요한지, 월수입 대비 얼마인지 작성
- **관심도 점수** — 0~100점, 시간이 지날수록 자동 감소. 관심이 식으면 자연히 사라짐
- **대시보드** — 이번달 절약한 금액, 포기한 아이템 수 통계
- **공유 확장** — 쇼핑몰에서 상품 URL을 공유하면 앱으로 바로 추가
- **푸시 알림** — 관심도가 10점 이하로 떨어지면 "아직 사고 싶으세요?" 알림

## 기술 스택

| 역할 | 기술 |
|------|------|
| 앱 | React Native + Expo |
| 라우팅 | Expo Router |
| 인증 | Firebase Auth (Google) |
| 데이터베이스 | Firebase Firestore |
| 푸시 알림 | Expo Notifications |
| 공유 | Expo Share Extension |

## 시작하기

```bash
# 의존성 설치
npm install

# 개발 서버 시작
npx expo start
```

> Firebase 설정 필요: `.env.local`에 Firebase 키 입력, `GoogleService-Info.plist` (iOS) 및 `google-services.json` (Android) 배치

## 관심도 점수 로직

- 아이템 추가 시 100점으로 시작
- 매일 자동 감소 (느리게: 30일, 보통: 14일, 빠르게: 7일에 0점 도달)
- 상세 페이지에서 "아직 사고 싶어요" 클릭 시 +5점 (관심 신호)
- 10점 이하 → 알림 발송 → 3일 무응답 시 자동 포기 처리
